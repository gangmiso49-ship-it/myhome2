# 그린테이블 UI/UX 기획서

## 1. 전체 디자인 콘셉트
- **콘셉트 명:** Nature in the City (도심 속 친환경 식탁)
- **방향성:** 복잡한 장식을 배제한 미니멀리즘(Minimalism)을 기반으로, 여백을 충분히 활용하여 시각적인 편안함과 깔끔함을 제공합니다. 고해상도의 신선한 식재료 및 메뉴 이미지를 돋보이게 하여 건강하고 세련된 브랜드 이미지를 구축합니다.

## 2. 메인 컬러와 보조 컬러
- **메인 컬러 (Primary):** `Forest Green (#2E7D32)` - 브랜드의 핵심인 '자연, 건강, 신뢰'를 상징합니다. 로고, 헤더, 주요 타이틀에 사용합니다.
- **보조 컬러 (Secondary):** `Warm Beige (#F8F5EE)` & `Soft White (#FAFAFA)` - 따뜻하고 깨끗한 배경색으로 사용하여 눈의 피로를 덜고 메인 컬러와 이미지들을 부드럽게 받쳐줍니다.
- **포인트 컬러 (Accent):** `Fresh Orange (#F28E2B)` - 식욕을 돋우고 주목도를 높이는 컬러로, '주문하기', '문의하기' 등 주요 CTA (Call To Action) 버튼에 사용하여 전환율을 높입니다.

## 3. 폰트 스타일
- **기본 폰트:** `Pretendard (프리텐다드)`
- **선정 이유:** 모던하고 세련된 산세리프(San-serif) 서체로, 다양한 디바이스 환경에서도 뛰어난 가독성을 자랑합니다.
- **활용 가이드:**
  - **Title (제목):** Pretendard Bold / ExtraBold (시각적 위계를 명확히 하여 핵심 메시지 전달)
  - **Body (본문):** Pretendard Regular / Medium (장문의 텍스트도 편안하게 읽을 수 있도록 자간과 행간 160% 적용)

## 4. 메인 화면 구성 (Home)
- **GNB (글로벌 네비게이션 바):** 상단 고정(Sticky). 로고(좌측), 메뉴(중앙), 문의하기 버튼(우측) 배치. 스크롤 시 배경색이 화이트로 전환되며 가시성 확보.
- **Hero Section:** 화면을 꽉 채우는 고화질 메인 이미지/비디오 루프(신선한 샐러드 조리 과정 등) 위에 브랜드 캐치프레이즈와 '메뉴 보러가기' CTA 버튼 중앙 배치.
- **Best Menu Section:** 가로 스와이프(Carousel) 형태의 UI를 적용하여 인기 메뉴를 노출.
- **Brand Value Section:** 3개의 주요 키워드(신선함, 친환경, 건강)를 아이콘과 짧은 텍스트로 직관적으로 전달.

## 5. 브랜드 소개 영역 구성 (About Us)
- **스토리텔링 UI:** 스크롤에 따라 요소가 페이드인(Fade-in) 되는 애니메이션을 적용하여 사용자가 브랜드 스토리에 몰입할 수 있도록 유도.
- **시각적 요소:** 텍스트 위주의 나열을 피하고, 산지 직송 식재료, 위생적인 주방 환경 등을 보여주는 넓은 레이아웃의 사진과 여백을 교차 배치(Z-Pattern 텍스트/이미지 배치)하여 세련미 강조.

## 6. 메뉴 소개 카드 구성 (Menu)
- **카드형 UI (Card Interface):** 깔끔한 그리드(Grid) 배열. 각 메뉴 카드는 모서리가 둥근(Round: 12px) 디자인으로 부드러운 인상 부여.
- **마이크로 인터랙션 (Hover Effect):** PC에서 메뉴 카드에 마우스를 올리면 부드러운 그림자(Drop Shadow)가 생기며 카드가 살짝 떠오르는 효과를 주어 클릭 유도.
- **정보 위계:** 
  1. 고해상도 누끼 이미지 (시각적 자극)
  2. 뱃지/태그 (예: [Best], [Vegan], [고단백])
  3. 메뉴명 (Bold)
  4. 핵심 영양정보 (칼로리, 단백질량 등 - 작은 아이콘과 함께 표기)

## 7. 친환경 가치 소개 영역 구성 (Eco-Friendly)
- **인포그래픽 활용:** 생분해성 용기, 사탕수수 종이, 콩기름 잉크 사용 등의 친환경 활동을 복잡한 텍스트 대신 픽토그램(Pictogram)과 인포그래픽으로 구성하여 직관적인 이해를 도움.
- **패럴랙스 스크롤 (Parallax Scroll):** 배경에는 친환경 숲이나 자연의 이미지를 고정하고 콘텐츠가 스크롤되게 하여 시각적인 청량감과 브랜드 가치를 극대화.

## 8. 문의하기 영역 구성 (Contact)
- **입력 폼 (Form):** 사용자의 입력 부담을 줄이기 위해 필수 입력 항목을 최소화. 
  - 주문 유형 선택(B2B 단체, B2C 정기구독 등)은 드롭다운 또는 라디오 버튼으로 직관적으로 구성.
- **입력 필드 디자인:** 선(Line) 위주의 깔끔한 인풋 박스(Input box) 디자인. 입력 포커스(Focus) 시 메인 컬러(Forest Green)로 활성화 테두리 표시.
- **플로팅 버튼 (FAB):** 화면 우측 하단에 카카오톡 1:1 상담하기 플로팅 아이콘을 고정 배치하여 언제든 빠른 상담이 가능하도록 접근성 향상.

## 9. 모바일 반응형 구성
- **모바일 최적화:** 브레이크포인트(Breakpoint)를 설정하여 스마트폰 화면에서는 1열(1-Column) 리스트 형태로 전환.
- **GNB 변경:** 모바일에서는 햄버거 메뉴(Hamburger Menu)로 접어두어 화면 공간 확보.
- **터치 영역 (Touch Target):** 모든 버튼과 링크는 모바일 사용자의 손가락 굵기를 고려하여 최소 44x44px 이상의 터치 영역을 확보하여 오터치 방지.
- **하단 고정 버튼:** 모바일에서는 스크롤이 길어지므로 '주문 문의하기' CTA 버튼을 화면 하단에 Sticky 형태로 고정하여 전환 유도.

## 10. 사용자 경험(UX)을 높이기 위한 요소
- **스켈레톤 로딩 (Skeleton Loading):** 이미지가 많은 메뉴 페이지에서 로딩 지연 시 빈 화면 대신 뼈대(Skeleton) UI를 먼저 보여주어 체감 로딩 시간을 단축.
- **접근성 강화 (Accessibility):** 모든 이미지에 대체 텍스트(Alt Text)를 제공하고, 배경색과 텍스트의 대비(Contrast)를 적절히 유지하여 누구나 편하게 볼 수 있도록 디자인.
- **매끄러운 탐색 (Smooth Scroll):** 앵커 링크 클릭 시 해당 섹션으로 부드럽게 스크롤링되는 효과를 주어 세련된 웹사이트 이용 경험 제공.
